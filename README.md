# euroscipy2017

## Setup

1. Create a new virtual environment.

```
virtualenv -p $(which python3.6) .venv
echo "source .venv/bin/activate" > .env  
source .env
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Launch Jupyter notebooks

```
jupyter notebook
```
